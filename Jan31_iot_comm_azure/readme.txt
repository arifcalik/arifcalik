Check this quickstart provided from Microsoft to send some data from the Device to the IoTHub in azure:
https://learn.microsoft.com/en-us/azure/iot-hub/c2d-messaging-python
You are going to simulate a microcomputer from a python script and then send that data to the cloud.
 
When you are done sending the data to IoThub.. Try to route the data in a storage account so the data can be saved somewhere.
